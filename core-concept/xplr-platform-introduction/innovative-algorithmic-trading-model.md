# Innovative Algorithmic Trading Model-Pooling Algorithm Model

XPLR's core trading algorithm is the **Pooling Algorithm Model**, designed to efficiently centralize NFT transaction matching and settlement:

- **Pooling Trades**: Unlike traditional direct matching between buyers and sellers, all NFTs are aggregated in a pool where both buyers and sellers interact directly with the pool, significantly improving liquidity.
- **Instant Settlement**: Transaction participants receive immediate results upon participation without waiting for transaction cycles to complete.
- **Enhanced Trading Speed and Efficiency**: Centralized trading reduces waiting time for matching.
- **Optimized Liquidity**: Attracts more market participants, reducing transaction execution time and costs.
