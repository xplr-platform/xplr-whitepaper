# Trading Reward

Users receive different rewards based on the transaction price range:

| Price Range | Trading Reward |
| :--- | :--- |
| $50-500 | 1.8% |
| $500-2,000 | 2.1% |
| $2,000-5,000 | 2.6% |
| $5,000-10,000 | 3.1% |
| $10,000-30,000 | 3.7% |
| $30,000-50,000 | 4.1% |

**Reservation Deposit**: When reserving an NFT, the account must have sufficient balance. If the reservation fails, assets will not be deducted; funds are only deducted upon successful reservation.
