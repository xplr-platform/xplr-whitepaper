# FAQ

**Q: What is XPLR?**
A: XPLR is the world's first comprehensive NFT trading platform based on algorithmic trading.

**Q: How does the Pooling Algorithm Model work?**
A: It aggregates all NFTs in a pool where buyers and sellers interact directly, ensuring instant settlement and continuous liquidity.

**Q: How do I earn rewards?**
